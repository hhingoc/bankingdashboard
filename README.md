# bankingdashboard
🏦 Dashboard Quản trị Tín dụng Ngân hàng

Một dashboard phân tích dữ liệu tín dụng ngân hàng được xây dựng bằng Python + Streamlit + Plotly, hỗ trợ theo dõi hiệu quả hoạt động, chất lượng tín dụng và cơ cấu danh mục thông qua giao diện trực quan và tương tác.
